Что такое таблица?

Таблица в базе данных — как комната в норке енота. В одной комнате хранятся только однотипные вещи. Нельзя смешивать студентов с книгами или товарами!

📦

Типы данных (какие полки бывают)

🔑 Счетчик Автоматически нумерует (1,2,3...) — для ID
📝 Текст Для имён, фамилий, адресов, названий
🔢 Число Для баллов, возраста, цен, количества
📅 Дата/время Для дат рождения, событий, года издания

Первичный ключ — главная комната

У каждой записи должен быть уникальный номер. Обычно это поле ID с типом "Счетчик". Он помогает не перепутать студентов с одинаковыми именами.

🚫

Нормализация — не бери лишнее!

В таблице должны быть только поля, относящиеся к её теме.

✅ Для таблицы "Студенты" подходят: Фамилия, Имя, Класс, Средний балл ❌ Не подходят: Название книги, Цена товара, Автор
