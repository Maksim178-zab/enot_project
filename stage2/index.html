🔨 КАК СОЗДАТЬ ЗАПРОС В КОНСТРУКТОРЕ

1
Сортировка по одному полю
В режиме таблицы → нажать на заголовок поля → выбрать «Сортировка по возрастанию/убыванию»
2
Многоуровневая сортировка
Данные → Сортировка → добавить уровень → выбрать поля и порядок
3
Запрос с условием
Создание → Конструктор запросов → добавить таблицу → в строке «Условие отбора» написать: "Жёлтый" или >80
4
Группировка
Конструктор запросов → кнопка «Итоги» (∑) → в строке «Группировка» выбрать нужную функцию (Count, Sum, Avg)
5
Вычисляемое поле
В пустой ячейке «Поле» написать: Вес_кг: [Вес]/1000

💡 СОВЕТЫ

  • В Access удобно создавать запросы через конструктор
  • После создания запроса сохрани его (Ctrl+S)
  • Для группировки нажми кнопку «Итоги» (∑)
  • Вычисляемые поля создаются в строке «Поле»: Вес_кг: [Вес]/1000
  • Проверь запросы на разных данных
