# 🦝 Не грузи енота

**Игровой квест по сортировке данных** для учащихся 10 класса.

## 🎯 О проекте

Это интерактивный образовательный проект, в котором ученики в игровой форме осваивают:

- Очистку данных от дубликатов и ошибок
- Создание таблиц в MS Access
- Сортировку и запросы к данным
- Подготовку к защите проекта

## 🚀 Как начать

1. Открой `index.html` в браузере
2. Выбери команду или создай новую
3. Проходи этапы квеста и зарабатывай бананы 🍌
